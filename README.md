# Golang Redis 
- Building Redis database using Golang

## Commands 
- `PING` -> `PONG`
- `SET name ahmed` -> `OK`
- `GET name` -> `ahmed`
- `HSET users num_1 ahmed` -> `OK`
- `HGET users num_1` -> `ahmed`
- `HGETALL users` -> `ahmed`
   
# How to run the project ?
- `go run *.go` to run redis server
- `redis-cli` to run redis client (here you put your commands)
